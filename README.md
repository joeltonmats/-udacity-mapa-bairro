## How to run ?

* Donwload or clone the project
* Go to project folder

<pre>
<code>
    $PROJECT_FOLDER npm install
</code>
</pre>

* After install the dependecies

<pre>
<code>
  $PROJECT_FOLDER  npm start
</code>
</pre>

* Open your browser on port 3000
